Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat struktur halaman web.

Contoh Struktur HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>Ini adalah paragraf.</p>
  </body>
</html>

HTML terdiri dari elemen-elemen seperti heading, paragraf, gambar, dan tautan.

Elemen HTML Populer

  • <h1> sampai <h6>: Digunakan untuk membuat judul (heading) dengan tingkat kepentingan berbeda.
  • <p>: Digunakan untuk menulis paragraf teks biasa.
  • <a>: Elemen tautan untuk membuat hyperlink ke halaman lain.
  • <img>: Menampilkan gambar di halaman web.
  • <ul> dan <li>: Untuk membuat daftar bullet.
  • <table>: Digunakan untuk membuat tabel.
  • <form>: Untuk membuat formulir input pengguna.

Cara Mulai Belajar HTML

Kamu bisa mulai menulis HTML dengan teks editor seperti Visual Studio Code atau Notepad++.

Simpan file dengan ekstensi .html, lalu buka dengan browser untuk melihat hasilnya.

Tips Pemula

  • Tulis HTML dengan rapi dan terstruktur.
  • Gunakan komentar <!-- komentar --> untuk memberi catatan.
  • Pelajari satu per satu elemen HTML secara mendalam.
